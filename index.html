    • Charge Pump

    • In order to achieve a voltage up to 25 V from voltage range of 1-3V, designed a charge pump system which has configurable stages.
    • The charge pump receives this 1-3 voltage input from a Pre-Amp which gets input of 10 mV-100 mV range from a MEMS sensor. Used Arduino for charge pump configuration.

    Speech Recognition

    • An audio file with the corresponding voice recording is stored in the specified path, which is later matched with the one being called, by using correlation and Fast Fourier Transform (FFT).
    • Used Matlab and CCS software.

    Security Bell

    • Designed a bell using light dependent register (LDR) which starts buzzing whenever there is a light obstruction to the sensor.
    • Components used were a battery,an LDR, an LED, two registers and one transitor.
